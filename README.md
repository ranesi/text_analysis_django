Text Analysis Advanced
=======

Setup:  
1. make your virtualenv (python 3.x)
2. `pip install -r requirements.txt`
3. after installing requirements, do the following in the python interpreter:  

`>> import nltk`  
`>> nltk.download()`  
*Should display the nltk menu*  
`Downloader> d`  
`Identifier> cmudict`  
  
That is it! Enjoy!?
